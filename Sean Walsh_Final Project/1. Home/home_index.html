    What is a CDN?

    CDN stands for a content delivery network (CDN).

    Read More

    This is a collection of global servers that caches and delivers content such as images, videos and Javascript files. When a user requests a web page, for instance, content on that page is delivered by the closest edge server rather than the origin server, thereby decreasing page load time.
